# 🎮 Урок 4.4: Вирівнювання по сітці — `snapToGrid()`

## Навіщо вирівнювати?

В оригінальній грі Battle City танк рухається по **сітці з тайлів**. Кожен тайл — 16 пікселів. Коли танк повертає, він "прилипає" до найближчої лінії сітки.

Без вирівнювання:
```
Танк їде вправо, X = 53.5
Гравець натиснув "вгору"
Танк повертається, але X = 53.5 — не кратне 16!
Танк "застрягне" між клітинками сітки!

      тайл 48    тайл 64
        │           │
        │  [ТАНК]   │  ← X=53.5, не влазить!
        │     ↑     │
```

З вирівнюванням:
```
Танк їде вправо, X = 53.5
Гравець натиснув "вгору"
snapToGrid → X округлюється до 48 (найближче кратне 16)
Танк акуратно стає в клітинку!

      тайл 48    тайл 64
        │           │
      [ТАНК]        │  ← X=48, ідеально!
        ↑           │
```

---

## Додаємо `snapToGrid()` до `Tank.js`

Додай цей метод в клас `Tank`, перед методом `move()`:

```javascript
/**
 * Вирівнює танк по тайловій сітці при повороті.
 *
 * Якщо танк повертає на горизонтальний рух (left/right) —
 * вирівнюємо Y до найближчого тайла.
 *
 * Якщо на вертикальний (up/down) — вирівнюємо X.
 *
 * @param {string} newDir — новий напрямок
 */
snapToGrid(newDir) {
    if (newDir === 'left' || newDir === 'right') {
        // Горизонтальний рух → Y має бути кратним TILE
        this.y = Math.round(this.y / TILE) * TILE;
    } else {
        // Вертикальний рух → X має бути кратним TILE
        this.x = Math.round(this.x / TILE) * TILE;
    }
}
```

---

## Розбираємо `Math.round(this.x / TILE) * TILE`

Ця формула округлює число до найближчого кратного `TILE` (16):

```
this.x = 53.5

Крок 1: 53.5 / 16 = 3.34375
Крок 2: Math.round(3.34375) = 3
Крок 3: 3 * 16 = 48                   ← результат!
```

Ще приклади:

| `this.x` | `÷ 16` | `round` | `× 16` | Результат |
|-----------|--------|---------|---------|-----------|
| 53.5 | 3.34 | 3 | 48 | 48 |
| 60.0 | 3.75 | 4 | 64 | 64 |
| 32.0 | 2.00 | 2 | 32 | 32 (без змін) |
| 7.8 | 0.49 | 0 | 0 | 0 |

## Коли спрацьовує snap?

Подивись ще раз на `move()`:

```javascript
if (dir !== this.direction) {
    this.snapToGrid(dir);   // ← ось тут!
    this.direction = dir;
}
```

Snap спрацьовує **тільки при зміні напрямку**. Якщо танк їде прямо без повороту — snap не потрібен.

```
Їдемо вправо (X = 53.5):
  натиснули D → dir === direction → БЕЗ snap → продовжуємо

Повертаємо вгору (X = 53.5):
  натиснули W → dir !== direction → SNAP! → X = 48, direction = 'up'
```
