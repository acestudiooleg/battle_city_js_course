# 🎮 Урок 14.2: Обробка меню та інтро музика

## Обробка клавіш у меню

```js
    /**
     * Обробник натискання клавіш у меню.
     * ↑/W — переміщує курсор на "1 PLAYER"
     * ↓/S — переміщує курсор на "2 PLAYERS"
     * Enter/Space — підтверджує вибір
     */
    function onKey(e) {
        // Якщо вже підтверджено — ігноруємо натискання
        if (confirmed) return;

        // Стрілка вгору або W → обрати "1 PLAYER"
        if (e.code === 'ArrowUp' || e.code === 'KeyW') {
            selected = 0;
            e.preventDefault();  // щоб сторінка не скролилась
        }

        // Стрілка вниз або S → обрати "2 PLAYERS"
        if (e.code === 'ArrowDown' || e.code === 'KeyS') {
            selected = 1;
            e.preventDefault();
        }

        // Enter або Space → підтвердити вибір та запустити гру
        if (e.code === 'Enter' || e.code === 'Space' || e.code === 'NumpadEnter') {
            confirmed = true;

            // Прибираємо обробник — більше не потрібен
            document.removeEventListener('keydown', onKey);

            // Запускаємо гру! selected+1 = кількість гравців (1 або 2)
            startGame(selected + 1);
            e.preventDefault();
        }
    }

    // Додаємо обробник клавіш
    document.addEventListener('keydown', onKey);
```

## Анімаційний цикл меню

```js
    /**
     * Цикл анімації меню.
     * Малює меню 60 разів на секунду, поки гравець не підтвердить вибір.
     * Коли confirmed = true — цикл зупиняється.
     */
    function menuLoop() {
        // Якщо гравець підтвердив — припиняємо малювання меню
        if (confirmed) return;

        // Малюємо титульний екран
        drawMenu();

        // Просимо браузер викликати menuLoop знову на наступному кадрі
        requestAnimationFrame(menuLoop);
    }

    // Запускаємо цикл меню!
    menuLoop();
```

## Запуск гри

```js
    /**
     * Запускає гру з обраною кількістю гравців.
     * Створює об'єкт Game та викликає start().
     *
     * Важливо: натискання клавіші у меню = "user gesture" для браузера.
     * Це розблоковує автовідтворення звуку (Audio.play()),
     * тому інтро музика зможе грати!
     *
     * @param {number} numPlayers - 1 або 2
     */
    function startGame(numPlayers) {
        // Створюємо гру з вибраною кількістю гравців
        const game = new Game(canvas, numPlayers);

        // Запускаємо ігровий цикл (update + render)
        game.start();

        // Зберігаємо в window — для дебагу через консоль
        window.game = game;
    }
```

## 🔊 Інтро музика

В `Game.start()` граємо інтро мелодію:

```js
start() {
    this.running = true;

    // Інтро музика!
    // Браузер дозволяє грати звук, бо гравець вже натиснув
    // клавішу в меню (це називається "user gesture")
    this.sound.play('intro', 0.5);

    // Запускаємо ігровий цикл
    requestAnimationFrame((t) => this._loop(t));
}
```

## 📖 Розбираємо: User Gesture та autoplay

Браузери **блокують** автоматичне відтворення звуку. Це правило безпеки, щоб сайти не "кричали" без дозволу.

Але є виняток: якщо звук відтворюється **у відповідь на дію користувача** (клік, натискання клавіші) — браузер дозволяє.

```
❌ Не працює:
   сторінка завантажилась → audio.play() → ЗАБЛОКОВАНО!

✅ Працює:
   сторінка завантажилась → гравець натиснув Enter → audio.play() → ГРАЄ!
```

Тому наша схема: **меню** → натискання Enter → **запуск гри** + **інтро музика**. Всі щасливі!
