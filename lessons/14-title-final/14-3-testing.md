# 🧪 Урок 14.3: Фінальне тестування

## Чек-лист перевірки

Перш ніж сказати "гра готова!", давай перевіримо **кожну механіку**. Відкрий гру і пройдись по цьому списку:

---

### 1. Титульний екран
- [ ] Логотип "BATTLE CITY" по центру
- [ ] Курсор-танк рухається між пунктами (↑↓ або W/S)
- [ ] Enter/Space запускає гру
- [ ] "1 PLAYER" → один танк, "2 PLAYERS" → два танки

### 2. Рух
- [ ] P1: WASD працює (4 напрямки)
- [ ] P2: Стрілки працюють (якщо 2 гравці)
- [ ] Танк не виходить за межі поля
- [ ] Танк не проходить крізь стіни та інші танки
- [ ] Гусениці анімуються при русі

### 3. Стрільба
- [ ] P1: Space стріляє
- [ ] P2: Enter стріляє
- [ ] Кулі летять у напрямку руху
- [ ] Кулі зникають за межами поля
- [ ] Кулдаун працює (не можна стріляти безперервно)

### 4. Стіни
- [ ] Цегла руйнується (2 тайли з боку влучання)
- [ ] Бетон НЕ руйнується
- [ ] Вода — танки не проїжджають, кулі пролітають
- [ ] Ліс — танки проїжджають під ним (ліс поверх)
- [ ] Анімація води (мерехтіння)

### 5. Штаб (Орел)
- [ ] Орел видно внизу по центру
- [ ] Захисна П-подібна цегляна стінка навколо
- [ ] Куля у штаб → орел змінюється на зруйнований → Game Over
- [ ] Великий вибух при знищенні

### 6. Вороги
- [ ] Вороги спавняться з 3 точок зверху
- [ ] Мигаюча зірка перед появою (2 сек)
- [ ] 4 типи працюють (basic, fast, power, armor)
- [ ] AI рухається і стріляє
- [ ] Максимум 4 одночасно на полі
- [ ] Куля гравця ↔ куля ворога — обидві знищуються

### 7. Стан гри
- [ ] P — пауза/відновлення
- [ ] R — рестарт (повернення до меню)
- [ ] Game Over: коли всі гравці втратили життя
- [ ] Game Over: анімація тексту знизу вгору
- [ ] Перемога: коли всі 20 ворогів знищені
- [ ] Перемога: overlay з текстом

### 8. Звуки
- [ ] Інтро мелодія при запуску
- [ ] Звук пострілу
- [ ] Звук вибуху (маленький та великий)
- [ ] Двигун при русі (петля)

### 9. Артефакти
- [ ] З'являються на полі (мигають)
- [ ] Підбираються при наїзді
- [ ] tank: +1 життя
- [ ] helmet: щит (мигання)
- [ ] star: швидший танк
- [ ] grenade: знищує всіх ворогів
- [ ] timer: вороги заморожені
- [ ] shovel: бетон навколо штабу

### 10. Графіка
- [ ] NES спрайти для танків (4 напрямки × 2 кадри)
- [ ] NES спрайти для тайлів
- [ ] Орел зі спрайтів
- [ ] Чіткі пікселі (без розмиття)
- [ ] Sidebar з іконками ворогів, лейблом IP, прапором

---

## 🐛 Що робити якщо знайшов баг?

1. **Відтвори баг** — зроби те саме ще раз, щоб зрозуміти коли він виникає
2. **Знайди файл** — подумай який клас відповідає за цю механіку
3. **Додай `console.log()`** — виведи значення змінних, щоб зрозуміти що не так
4. **Виправ і перевір** — зміни код і перевір що баг зник
