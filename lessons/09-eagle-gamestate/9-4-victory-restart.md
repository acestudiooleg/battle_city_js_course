# 🏆 Урок 9.4: Перемога та рестарт

## Умова перемоги

В Battle City гравець перемагає коли **всі 20 ворогів знищені** і на полі не залишилось жодного ворога. Поки у нас немає ворогів (це буде в уроці 10), але ми вже підготуємо систему підрахунку.

---

## 📊 Лічильники в main.js

Додаємо змінні для відстеження перемоги:

```js
// ─── Лічильники ─────────────────────────────────────────────────
let killedCount = 0;        // скільки ворогів знищено
const TOTAL_ENEMIES = 20;   // скільки всього ворогів у рівні
```

### Перевірка перемоги в `update()`:

```js
// В кінці функції update():

// Перемога: всі вороги знищені
// enemies — масив активних ворогів (поки пустий)
// spawnQueue — черга ворогів що ще не з'явились (поки пуста)
if (killedCount >= TOTAL_ENEMIES && enemies.length === 0) {
    victory = true;
}
```

## 🔄 Рестарт

В демо уроку 9 ми використовуємо найпростіший спосіб рестарту — **перезавантаження сторінки**:

```js
if (input.justRestart()) {
    location.reload();
}
```

Це повністю скидає стан гри: всі об'єкти створюються заново.
