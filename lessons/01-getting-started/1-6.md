## ▶️ Крок 1: Запускаємо через Live Server

1. Відкрий папку `lesson1` в VS Code (File → Open Folder)
2. Знайди файл `index.html` у списку файлів зліва
3. **Правий клік на `index.html`**
4. Вибери **"Open with Live Server"**
5. Браузер відкриється автоматично! 🌐

---

## 🔧 Альтернативні способи запуску

Якщо Live Server не працює:

### Через термінал VS Code:
```bash
npx live-server
```

### Через Python:
```bash
python -m http.server 8000
```
Потім відкрий `http://localhost:8000`

### Через Node.js:
```bash
npx http-server -p 8000
```
Потім відкрий `http://localhost:8000`

## 🔍 Крок 3: Дивимося консоль

1. Натисни **F12** (або правий клік → "Перевірити")
2. Перейди на вкладку **"Console"**
3. Ти побачиш повідомлення:
   - `🚀 Сторінка готова, запускаємо гру...`
   - `🎮 Гра "Танчики" запущена!`
   - `📐 Розмір аркуша: 800 x 600`
   - `🔲 Розмір клітинки: 32 пікселів`

Консоль — це секретний щоденник нашої гри! 📓 Якщо щось піде не так, помилки теж будуть тут.
