# 🏗️ Урок 1.3: Створюємо HTML файл

## Що таке HTML?

HTML — це як скелет нашого сайту! 🦴

Уяви, що ти будуєш будинок. Спочатку потрібно зробити каркас — стіни, дах, вікна. HTML — це такий каркас для нашого сайту. Він каже браузеру, що і де показувати на екрані!

---

## 📝 Крок 1: Створюємо файл

В VS Code створи новий файл з назвою **`index.html`**

Напиши базову структуру:

```html
<!DOCTYPE html>
<html lang="uk">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Танчики - Урок 1: Готуємося до гри!</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>🎮 Танчики - Урок 1</h1>
    <div class="lesson-info">
        <h2>Готуємося до гри!</h2>
        <p>Сьогодні ми зробимо:</p>
        <ul style="text-align: left;">
            <li>✅ Створимо HTML файл</li>
            <li>✅ Додамо Canvas для малювання</li>
            <li>✅ Зробимо все красивим</li>
            <li>✅ Підготуємо основу для гри</li>
        </ul>
    </div>

    <canvas id="gameCanvas" width="800" height="600"></canvas>

    <script type="module" src="main.js"></script>
</body>
</html>
```

---

## 🎨 Що таке Canvas?

**Canvas — це як аркуш паперу, на якому ми будемо малювати нашу гру!** 🎨

Уяви, що у тебе є великий аркуш розміром 800 на 600 маленьких квадратиків (пікселів). На цьому аркуші ми малюватимемо танки, стіни, кулі — все!

```html
<canvas id="gameCanvas" width="800" height="600"></canvas>
```

- `id="gameCanvas"` — це ім'я нашого аркуша (щоб JavaScript знав, про який аркуш ми говоримо)
- `width="800"` — ширина 800 пікселів
- `height="600"` — висота 600 пікселів

---

## 🧠 Що таке `<script type="module">`?

**JavaScript — це мозок нашої гри!** 🧠

Рядок `<script type="module" src="main.js"></script>` підключає файл з кодом гри. Слово `module` означає, що ми зможемо розділяти код на окремі файли — як розкласти іграшки по коробках!
