# 📁 Урок 1.2: Створюємо папку для гри

## Що це означає?

Уяви собі, що ми створюємо спеціальну коробку для іграшок! 🎁

Ми створимо папку на комп'ютері, де будемо зберігати всі файли нашої гри. Все буде в одному місці і не загубиться!

---

- Урок 1: папка `lesson1` (перший рівень)
- Урок 2: папка `lesson2` (другий рівень)
- І так далі...

**Правило:** Кожен наступний урок копіює файли з попереднього і розширює їх новими можливостями — як нові здібності в грі! 🆙

---

## 📂 Крок 1: Створюємо папку

**Створи нову папку** з назвою `lesson1`:

### Windows
1. Відкрий Провідник (натисни **Win + E**)
2. Знайди зручне місце (наприклад, `C:\Projects\`)
3. Правий клік → "Створити" → "Папку"
4. Назви її `lesson1`

### macOS
1. Відкрий Finder
2. Знайди зручне місце (наприклад, `~/Documents/Projects/`)
3. **Command + Shift + N** для нової папки
4. Назви її `lesson1`

### Linux
1. Відкрий файловий менеджер
2. Знайди зручне місце (наприклад, `~/Projects/`)
3. **Ctrl + Shift + N** для нової папки
4. Назви її `lesson1`
