# 🔊 Урок 12.2: Відтворення звуків

## Метод `play()` — грати звук

Коли танк стріляє, ми хочемо сказати: "Грай звук пострілу!". Додаємо метод:

```js
/**
 * Відтворює звук за ключем.
 *
 * Чому clone? Уяви: танк стріляє двічі швидко. Перший звук ще грає,
 * а другий вже потрібен. Якщо використати той самий Audio — він перебʼє
 * перший. Тому ми клонуємо (копіюємо) — і кожен звук грає незалежно.
 *
 * @param {string} key    - ключ звуку ('shoot', 'explodeS', ...)
 * @param {number} volume - гучність (від 0.0 до 1.0, за замовч. 0.6)
 */
play(key, volume = 0.6) {
    // Якщо звуки вимкнені — нічого не робимо
    if (!this.enabled) return;

    // Знаходимо оригінальний Audio за ключем
    const src = this.sounds[key];

    // Якщо такого звуку немає — виходимо
    if (!src) return;

    // Клонуємо Audio — створюємо незалежну копію
    const clone = src.cloneNode();

    // Встановлюємо гучність для цієї копії
    clone.volume = volume;

    // Граємо! .catch(() => {}) — ігноруємо помилки
    // (браузер може заблокувати звук до першого кліку)
    clone.play().catch(() => {});
}
```

## Метод `setEngineSound()` — двигун

Двигун — особливий звук. Він **грає постійно**, поки танк рухається. Як тільки зупинився — двигун замовкає:

```js
/**
 * Запускає або зупиняє звук двигуна.
 * Двигун грає безперервно (loop=true) поки танк рухається.
 *
 * @param {boolean} moving - true = танк рухається, false = стоїть
 */
setEngineSound(moving) {
    // Якщо звуки вимкнені — нічого не робимо
    if (!this.enabled) return;

    // Беремо Audio двигуна
    const eng = this.sounds.engine;
    if (!eng) return;

    if (moving) {
        // Танк рухається — запускаємо двигун
        // .play() не перезапускає, якщо вже грає (розумний Audio!)
        eng.play().catch(() => {});
    } else {
        // Танк зупинився — зупиняємо двигун
        eng.pause();

        // Перемотуємо на початок — щоб наступного разу
        // звук почався з початку, а не з середини
        eng.currentTime = 0;
    }
}
```

## Метод `setEnabled()` — вимкнути/ввімкнути

```js
/**
 * Вмикає або вимикає всі звуки.
 * Якщо вимикаємо — зупиняємо все що грає.
 *
 * @param {boolean} value - true = звуки ввімкнені
 */
setEnabled(value) {
    this.enabled = value;

    if (!value) {
        // Проходимо по ВСІХ звуках і зупиняємо кожен
        // Object.values() — масив усіх значень об'єкта
        Object.values(this.sounds).forEach((s) => s.pause());
    }
}
```

## 📖 Розбираємо: чому `.catch(() => {})`?

Браузери мають правило: **звук не може грати, поки користувач не натисне будь-що на сторінці**. Це зроблено, щоб сайти не "кричали" звуками без дозволу.

Якщо ми спробуємо `audio.play()` без кліку — браузер поверне **помилку** (Promise rejected). `.catch(() => {})` каже: "Якщо помилка — просто ігноруй її".

```js
// Без catch — помилка в консолі
audio.play(); // 🔴 Uncaught Promise rejected

// З catch — тиша, ніяких помилок
audio.play().catch(() => {}); // ✅ Все окей
```
