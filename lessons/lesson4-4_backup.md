# 4.4: Фізика матеріалів та міцність стін 🧱🏗️

## Що ми будемо робити? 🚀

Сьогодні ми перетворимо наше ігрове поле на інтерактивне середовище. До цього моменту всі стіни зникали від одного пострілу. Але в реальних "Танчиках" цегла руйнується поступово, а бетон — майже неможливо пробити. Ми налаштуємо систему здоров'я (HP) для кожного типу блоку та пов'яжемо її з візуальними ефектами, які ми створили раніше.

## 🔧 Оновлення класу GameField.js

Відкрийте файл `GameField.js`. Ми додамо логіку "зношування" матеріалів та навчимо поле правильно реагувати на пошкодження.

### 🪵 Крок 1: Система міцності (HP)

Ми маємо чітко розділити типи стін. Дерево має руйнуватися миттєво, цегла — витримувати кілька влучань, а бетон — служити надійним укриттям. Це створює тактичну глибину: гравець буде знати, де можна сховатися, а де його швидко дістануть.

// ВСТАВИТИ КОД НАЛАШТУВАННЯ HP МАТЕРІАЛІВ ТУТ

```javascript
// Встановлюємо здоров'я залежно від матеріалу (у методі createRandomWalls)
let health, maxHealth;
switch (material) {
  case 'wood':
    health = maxHealth = 1;
    break;
  case 'brick':
    health = maxHealth = 3;
    break;
  case 'concrete':
    health = maxHealth = 10;
    break;
  default:
    health = maxHealth = 1;
}
```

### 🎨 Крок 2: Візуальний фідбек руйнувань

Щоб гравець бачив прогрес руйнування, ми оновимо метод малювання. Стіни, які втратили здоров'я, повинні змінювати свій колір або отримувати "тріщини". Ми реалізуємо це через зміну відтінку залежно від відсотка здоров'я, що залишилося.

// ВСТАВИТИ ЛОГІКУ ЗМІНИ КОЛЬОРУ ВІД HP ТУТ

```javascript
/**
     * Отримання кольору стіни залежно від матеріалу та здоров'я
     */
  getWallColor(wall) {
      const healthPercent = wall.health / wall.maxHealth;

      switch (wall.material) {
          case 'wood': return healthPercent > 0.5 ? '#8B4513' : '#654321';
          case 'brick':
              if (healthPercent > 0.66) return '#CD5C5C';
              if (healthPercent > 0.33) return '#B22222';
              return '#8B0000';
          case 'concrete': return '#708090';
          default: return '#7f8c8d';
      }
  }
```

### 🔨 Крок 3: Обробка пошкоджень

Ми додамо метод `damageWall`, який буде викликатися менеджером колізій. Цей метод не тільки зменшує HP, а й перевіряє, чи не час видалити стіну та створити в ігровому лозі повідомлення про успішну деструкцію.

// ВСТАВИТИ МЕТОД damageWall ТУТ

```javascript
/**
     * Пошкодження стіни
     * @returns {boolean} - true якщо стіна повністю зруйнована
     */
    damageWall(wall, damage = 1) {
        wall.health -= damage;

        if (wall.health <= 0) {
            const index = this.walls.indexOf(wall);
            if (index > -1) {
                this.walls.splice(index, 1);
                return true;
            }
        }
        return false;
    }
```

## 🎮 Нові механіки поля

### 🧱 Стратегія цегли:

Цегляна стіна тепер має 3 одиниці здоров'я. Це означає, що гравець може використовувати її як тимчасове укриття, але він має бачити, як вона "тане" під вогнем ворога.

### 🛡️ Бетонний щит:

Бетон має 10 одиниць здоров'я. Це робить його майже нездоланним для звичайної зброї. Це стимулює гравця шукати обхідні шляхи, а не просто стріляти в стіну.
