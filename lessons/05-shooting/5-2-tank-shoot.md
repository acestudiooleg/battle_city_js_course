# 🎮 Урок 5.2: Tank.shoot() — стрільба з кулдауном

## Вчимо танк стріляти!

Тепер додамо до класу `Tank` метод `shoot()` — він створюватиме кулю перед дулом танка.

---

## Що таке кулдаун?

**Кулдаун** (cooldown) — це мінімальна затримка між пострілами. Без нього гравець міг би стріляти 60 разів на секунду (кожен кадр)!

```
Кулдаун = 400 мс:

Постріл!     Можна!      Постріл!     Можна!
  │  400 мс   │            │  400 мс   │
──●────────────●────────────●────────────●──→ час
  │            │            │
 shoot()     shoot()      shoot()
 ✅ OK        ❌ ні        ✅ OK
              ↑ тільки 200мс пройшло
```

---

## Додаємо властивості для стрільби

Відкрий `Tank.js` і додай нові властивості в конструктор (після `this.animTimer`):

```javascript
        // ─── Стрільба ──────────────────────────────────────────
        this.bullets = [];         // масив куль цього танка
        this.shootCooldown = 400;  // мінімальна затримка між пострілами (мс)
        this.lastShotTime = 0;     // час останнього пострілу (мс)
```

---

## Додаємо метод `shoot()`

Додай імпорт `Bullet` на початку `Tank.js`:

```javascript
import { Bullet } from './Bullet.js';
```

Потім додай метод `shoot()` після `move()`:

```javascript
    /**
     * Створює кулю перед дулом танка.
     * Якщо кулдаун не закінчився — повертає null.
     *
     * @param {number} now   — поточний час (Date.now())
     * @param {string} owner — хто стріляє: 'player' або 'enemy'
     * @returns {Bullet|null} — нова куля або null
     */
    shoot(now, owner) {
        if (!this.alive) return null; // мертвий не стріляє

        // Перевіряємо кулдаун: чи пройшло достатньо часу з останнього пострілу?
        if (now - this.lastShotTime < this.shootCooldown) return null;

        this.lastShotTime = now; // запам'ятовуємо час цього пострілу

        // Центр танка
        const cx = this.x + this.width / 2;  // центр X
        const cy = this.y + this.height / 2; // центр Y

        // Стартова позиція кулі — перед дулом (за межами корпусу)
        const offsets = {
            up:    { bx: cx,                    by: this.y - 2 },
            down:  { bx: cx,                    by: this.y + this.height + 2 },
            left:  { bx: this.x - 2,            by: cy },
            right: { bx: this.x + this.width + 2, by: cy },
        };

        const { bx, by } = offsets[this.direction]; // позиція кулі

        // Створюємо кулю та додаємо до масиву
        const bullet = new Bullet(bx, by, this.direction, owner);
        this.bullets.push(bullet); // зберігаємо в масиві танка

        return bullet; // повертаємо для зовнішнього використання
    }
```

---

## Розбираємо код

### `Date.now()` — поточний час

`Date.now()` повертає кількість мілісекунд з 1 січня 1970 року. Це просто великий числовий "штамп часу":

```javascript
Date.now() // наприклад: 1711234567890
```

### Перевірка кулдауну

```javascript
if (now - this.lastShotTime < this.shootCooldown) return null;
```

Якщо різниця між `now` та `lastShotTime` менша за 400 мс — занадто рано, виходимо.

### Позиція кулі `offsets`

Куля має з'являтися **перед дулом**, а не в центрі танка:

```
       UP                           RIGHT

       ■ ← куля (by = y - 2)       ┌──────┐
    ┌──┬──┐                        │      ├──■ ← куля (bx = x + width + 2)
    │  │  │                        │      │
    │  └──│                        └──────┘
    └─────┘
```

`+2` — невеликий зсув, щоб куля не перетиналася з корпусом танка.

### `this.bullets.push(bullet)`

Кожен танк має **свій масив куль**. Коли танк стріляє, куля додається сюди. Це зручно — ми завжди знаємо, кому належить куля.
