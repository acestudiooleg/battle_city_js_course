## Як працює куля?

Куля — це маленький об'єкт (4×4 пікселі), який:
1. **Створюється** перед дулом танка
2. **Летить** у напрямку пострілу з постійною швидкістю
3. **Зникає** коли вилітає за межі поля

```
Танк стріляє вгору:

        ■ ← куля (4×4)
        │
        │  летить вгору
        │
    ┌──┬┴┬──┐
    │  │ │  │
    │  │ │  │
    │  └─┘  │
    └───────┘
```

## Створюємо `Bullet.js`

Створи новий файл `Bullet.js`:

```javascript
// Bullet.js — клас кулі
// Урок 5: Стрільба

import { FIELD_W, FIELD_H } from './constants.js';

/**
 * Клас Bullet — снаряд танка.
 * Летить у заданому напрямку з постійною швидкістю.
 * Деактивується при виході за межі поля.
 */
export class Bullet {
    /**
     * Створює нову кулю.
     * @param {number} fx        — X-позиція центру кулі (координати поля)
     * @param {number} fy        — Y-позиція центру кулі (координати поля)
     * @param {string} direction — напрямок польоту: 'up', 'down', 'left', 'right'
     * @param {string} owner     — хто стріляв: 'player' або 'enemy'
     */
    constructor(fx, fy, direction, owner) {
        this.x = fx;           // позиція центру X
        this.y = fy;           // позиція центру Y
        this.width  = 4;       // ширина кулі (4px)
        this.height = 4;       // висота кулі (4px)

        this.direction = direction; // напрямок польоту
        this.owner = owner;        // 'player' або 'enemy'
        this.speed = 5;            // швидкість (пікселів за кадр)
        this.active = true;        // чи активна куля (false = знищена)

        // Обчислюємо вектор руху (vx, vy) з напрямку
        // Це числа, які додаються до x та y кожен кадр
        const dirs = {
            up:    { vx:  0, vy: -1 },  // вгору: Y зменшується
            down:  { vx:  0, vy:  1 },  // вниз: Y збільшується
            left:  { vx: -1, vy:  0 },  // вліво: X зменшується
            right: { vx:  1, vy:  0 },  // вправо: X збільшується
        };

        // vx/vy = напрямок × швидкість
        this.vx = dirs[direction].vx * this.speed; // горизонтальна швидкість
        this.vy = dirs[direction].vy * this.speed;  // вертикальна швидкість
    }

    /**
     * Оновлює позицію кулі (викликається кожен кадр).
     * Якщо куля вилетіла за межі поля — деактивуємо.
     */
    update() {
        if (!this.active) return;

        // Рухаємо кулю
        this.x += this.vx; // додаємо горизонтальну швидкість
        this.y += this.vy; // додаємо вертикальну швидкість

        // Перевіряємо, чи куля ще на полі
        if (this.x < 0 || this.x > FIELD_W ||
            this.y < 0 || this.y > FIELD_H) {
            this.active = false; // куля вилетіла — деактивуємо!
        }
    }

    /**
     * Малює кулю на Canvas (білий квадратик 4×4).
     * @param {CanvasRenderingContext2D} ctx — контекст для малювання
     * @param {number} ox — зсув поля по X (FIELD_X)
     * @param {number} oy — зсув поля по Y (FIELD_Y)
     */
    render(ctx, ox, oy) {
        if (!this.active) return; // неактивну кулю не малюємо

        ctx.fillStyle = '#fcfcfc'; // білий колір (NES white)

        // Малюємо від центру: зсуваємо на половину ширини/висоти
        ctx.fillRect(
            Math.round(this.x - this.width / 2)  + ox, // screenX
            Math.round(this.y - this.height / 2) + oy, // screenY
            this.width,   // 4px
            this.height   // 4px
        );
    }
}
```

## Розбираємо код

### Вектор руху `vx, vy`

Замість перевіряти напрямок кожен кадр у `update()`, ми **один раз** обчислюємо вектор руху в конструкторі:

```
direction = 'up'   → vx = 0, vy = -5   (кожен кадр: y -= 5)
direction = 'right' → vx = 5, vy = 0   (кожен кадр: x += 5)
```

Це елегантний прийом — `update()` стає дуже простим:
```javascript
this.x += this.vx; // одна операція замість switch!
this.y += this.vy;
```

### `this.active` — стан кулі

Куля не видаляється одразу — вона стає **неактивною** (`active = false`). Пізніше ми очистимо масив від неактивних куль (урок 5.5).

### Малювання від центру

Координати кулі `(x, y)` — це її **центр**. Але `fillRect` малює від лівого верхнього кута. Тому зсуваємо на половину розміру:

```
         x
         │
    ┌────┼────┐
    │    │    │  width = 4
    ├────●────┤  ← центр (x, y)
    │    │    │
    └─────────┘
         │
  fillRect(x - 2, y - 2, 4, 4)
```
