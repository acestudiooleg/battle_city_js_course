# ⭐ Урок 13.2: Спавн артефактів

## Коли і де з'являються?

В NES Battle City артефакти з'являються:
- **Коли:** Випадково кожні **15–30 секунд**
- **Де:** У випадковому **вільному** місці на полі
- **Скільки:** Тільки **один** одночасно (новий не з'явиться, поки не підберуть старий)

---

## 🔧 Додаємо в Game.js

### Властивості в конструкторі

```js
// В конструкторі Game додаємо:

/** Активний артефакт на полі (тільки один одночасно) */
this.powerUp = null;

/**
 * Таймер до наступного спавну артефакту (мс).
 * Перший з'явиться через 10–20 секунд від початку гри.
 * Math.random() → випадкове число 0..1, * 10000 → 0..10000 мс
 */
this.powerUpTimer = 10000 + Math.random() * 10000;

// Ефекти артефактів
this.freezeTimer  = 0;     // скільки мс ще заморожені вороги
this.shovelTimer  = 0;     // скільки мс ще бетон навколо штабу
this.shovelActive = false; // чи зараз активний ефект лопати
```

---

### Метод `_updatePowerUps()`

Цей метод робить 3 речі:
1. Спавнить новий артефакт коли таймер = 0
2. Оновлює мигання активного артефакту
3. Перевіряє чи гравець підібрав артефакт

```js
/**
 * Оновлення системи артефактів.
 * Викликається кожен кадр з _update().
 *
 * @param {number} dt - deltaTime (мс)
 */
_updatePowerUps(dt) {
    // ─── 1. Спавн нового артефакту ───────────────────
    // Спавнимо тільки якщо зараз немає активного
    if (!this.powerUp) {
        // Зменшуємо таймер на кількість мілісекунд з минулого кадру
        this.powerUpTimer -= dt;

        // Коли таймер дійшов до нуля — час спавнити!
        if (this.powerUpTimer <= 0) {
            this._spawnPowerUp();
            // Наступний артефакт — через 15–30 секунд
            this.powerUpTimer = 15000 + Math.random() * 15000;
        }
    }

    // ─── 2. Оновлення мигання ────────────────────────
    if (this.powerUp) {
        this.powerUp.update(dt);
    }

    // ─── 3. Перевірка підбирання гравцем ─────────────
    // Якщо гравець живий і артефакт існує...
    if (player.alive && !player.isRespawning && this.powerUp && this.powerUp.active) {
        const pu = this.powerUp;

        // Перевіряємо перетин прямокутників (AABB)
        // Гравець і артефакт перетинаються якщо:
        // - лівий край гравця < правий край артефакту
        // - правий край гравця > лівий край артефакту
        // - верхній край гравця < нижній край артефакту
        // - нижній край гравця > верхній край артефакту
        if (player.x < pu.x + pu.width &&
            player.x + player.width > pu.x &&
            player.y < pu.y + pu.height &&
            player.y + player.height > pu.y) {
            // Перетин! Гравець підібрав артефакт
            this._collectPowerUp(player, pu);
        }
    }

    // ─── 4. Таймери ефектів ──────────────────────────
    // Заморозка (timer артефакт) — відраховуємо час
    if (this.freezeTimer > 0) {
        this.freezeTimer -= dt;
    }

    // Лопата (shovel артефакт) — бетон навколо штабу
    if (this.shovelActive && this.shovelTimer > 0) {
        this.shovelTimer -= dt;
        // Час вийшов — повертаємо цегляну стінку
        if (this.shovelTimer <= 0) {
            this.shovelActive = false;
            this.field.unfortifyEagle(); // повертаємо цеглу
        }
    }
}
```

### Метод `_spawnPowerUp()` — знайти вільне місце

```js
/**
 * Створює артефакт у випадковому вільному місці.
 * Робить до 50 спроб знайти місце, де:
 * - Немає стін, води, лісу
 * - Немає танків
 */
_spawnPowerUp() {
    // Робимо до 50 спроб (щоб не зациклитись)
    for (let attempt = 0; attempt < 50; attempt++) {
        // Випадкова позиція, вирівняна по блоках (32px = 2 тайли)
        // Math.random() * 12 → число від 0 до 11.99
        // Math.floor() → ціле число від 0 до 11
        // * 2 → парне число від 0 до 22
        const tx = Math.floor(Math.random() * 12) * 2;
        const ty = Math.floor(Math.random() * 12) * 2;

        // Перетворюємо тайлову координату в піксельну
        const fx = tx * TILE;  // наприклад: 4 * 16 = 64
        const fy = ty * TILE;

        // Перевіряємо: чи може танк стояти тут?
        // (це означає: немає стін, бетону, води)
        const testTank = { width: TANK_SIZE, height: TANK_SIZE };
        if (!this.field.canTankMove(testTank, fx, fy)) continue;

        // Перевіряємо: чи немає танків у цій позиції?
        let occupied = false;
        const allTanks = [player, ...this.enemies];
        for (const t of allTanks) {
            // Пропускаємо мертвих
            if (!t.alive) continue;
            // Перевіряємо перетин
            if (fx < t.x + t.width && fx + TANK_SIZE > t.x &&
                fy < t.y + t.height && fy + TANK_SIZE > t.y) {
                occupied = true;
                break;  // знайшли зайняте місце — вихід з циклу
            }
        }
        // Якщо зайнято — пробуємо знову (continue → наступна ітерація)
        if (occupied) continue;

        // Місце вільне! Створюємо артефакт
        this.powerUp = new PowerUp(fx, fy, PowerUp.randomType());
        return;  // Виходимо з функції — артефакт створено
    }
    // Якщо за 50 спроб не знайшли — не страшно, спробуємо пізніше
}
```

## 📖 Розбираємо: `continue` та `break`

В циклах `for` є два важливі слова:

```js
for (let i = 0; i < 10; i++) {
    if (i === 3) continue;  // пропустити i=3, перейти до i=4
    if (i === 7) break;     // зупинити весь цикл

    console.log(i); // Виведе: 0, 1, 2, 4, 5, 6
}
```

- **`continue`** — "пропусти цю ітерацію, перейди до наступної"
- **`break`** — "зупини весь цикл прямо зараз"
