# 4.3: Деструкція: Зв'язок кулі та вибуху 🎯

## Що ми будемо робити? 🚀

Сьогодні ми перетворимо "математичні" зіткнення на видовищні події. Наша мета — навчити систему колізій не просто мовчки видаляти кулі, а й віддавати наказ головному класу гри створити вибух саме в точці контакту. Ми зробимо гру "соковитою": тепер кожне влучання буде мати миттєвий візуальний фідбек.



## 🔧 Оновлення класу CollisionManager.js

Відкрийте файл `CollisionManager.js`. Ми зосередимося на методі, який обробляє влучання куль у стіни, а також на зіткненнях танків.

### 💥 Крок 1: Вибухи при влучанні в стіни
Зараз ваші кулі просто зникають, торкаючись перешкод. Нам потрібно додати виклик методу `createExplosion`. Але є нюанс: ми маємо запитати у ігрового поля, який саме тип вибуху підходить для цієї стіни (дерево, цегла чи бетон).

```javascript
// Отримуємо тип вибуху залежно від матеріалу стіни
  const explosionType = gameField.getWallExplosionType ? 
                        gameField.getWallExplosionType(wall) : 'small';

  // Створюємо вибух у точці контакту через посилання на гру
  if (this.game) {
    this.game.createExplosion(bullet.x, bullet.y, explosionType);
  }
```

### 🚜 Крок 2: Вибухи при влучанні в танки
Коли гравець влучає у ворога (або навпаки), це має супроводжуватися яскравим спалахом. Ми додамо логіку, яка створює середній або великий вибух залежно від того, чи танк був просто пошкоджений, чи повністю знищений.

```javascript
// Створюємо вибух у центрі танка при влучанні
    if (this.game) {
      const type = enemy.getHealth() <= 25 ? 'large' : 'medium';
      this.game.createExplosion(
        enemy.x + enemy.width / 2, 
        enemy.y + enemy.height / 2, 
        type
      );
    }
```

### 🛡 Крок 3: Захист штабу
Штаб — найважливіший об'єкт. Його знищення має бути найпотужнішим ефектом у грі. Ми додамо створення великого (`large`) вибуху при попаданні ворожої кулі в базу.

```javascript
// Створюємо потужний вибух при знищенні штабу
    if (this.game) {
      this.game.createExplosion(bullet.x, bullet.y, 'large');
    }
```

## 🎮 Логіка взаємодії

### 📍 Координати влучання:
Ми передаємо в метод `createExplosion` координати самої кулі (`bullet.x`, `bullet.y`). Це важливо, бо вибух має статися саме там, де "ніс" кулі торкнувся об'єкта, а не в центрі танка чи стіни.

### 🧬 Динамічні типи:
Завдяки тому, що ми передаємо `explosionType` з властивостей стіни, система автоматично підхопить потрібні налаштування з класу `Explosion`:
- Влучив у дерево? Отримуй дрібні тріски.
- Влучив у бетон? Отримуй яскраві іскри та дим.

## 🚀 Що ми отримали?

- ✅ Кожне влучання в грі тепер візуально підтверджується.
- ✅ Гравець одразу розуміє, куди він влучив, навіть не дивлячись на ігровий лог.
- ✅ Система колізій стала повноцінним "режисером" бойових сцен.
