# ✨ Урок 10.2: Spawn-анімація

## Як з'являються вороги в NES?

В оригінальній грі ворог **не з'являється миттєво**. Спочатку на місці появи мигає **зірка** протягом 2 секунд, а потім з'являється танк. Це дає гравцю час підготуватись!

```
Мигання зірки (2 сек) → Танк з'являється
     ✦ → ✧ → ✦ → ✧       → 🔴
```

---

## 🌟 Як працює анімація

В конструкторі Enemy ми вже додали:
```js
this.spawnFlash = true;       // чи зараз мигає зірка
this.spawnTimer = 2000;       // 2 секунди
this.flashFrame = 0;          // кадр мерехтіння
```

Під час spawn-анімації ворог:
- **Не рухається** (стоїть на місці)
- **Не стріляє** (ще не народився)
- **Не може отримати пошкодження** (невразливий)
- **Малюється як зірка** (мигаючий ефект)

---

## 🎨 Малювання зірки

Додаємо метод `render()` в Enemy:

```js
/**
 * Малювання ворога
 * Якщо spawn-анімація — малюємо мигаючу зірку
 * Якщо ні — малюємо танк як зазвичай (через super.render)
 */
render(ctx, ox, oy) {
    // Мертвого ворога не малюємо
    if (!this.alive) return;

    // ─── Spawn-анімація: мигаюча зірка ────────────────────────
    if (this.spawnFlash) {
        // Позиція на Canvas
        const dx = Math.round(this.x) + ox;
        const dy = Math.round(this.y) + oy;

        // Малюємо тільки парні кадри (створює ефект мигання)
        // flashFrame йде 0,1,2,3,4,5,6,7,0,1,...
        // Малюємо коли flashFrame < 4 (половину часу видно)
        if (this.flashFrame < 4) {
            ctx.save();
            ctx.strokeStyle = '#fcfcfc'; // біла обведка
            ctx.lineWidth = 2;

            // Малюємо коло (зірку) в центрі позиції танка
            const r = this.width / 2 + 4; // радіус трохи більший за танк
            ctx.beginPath();
            ctx.arc(
                dx + this.width / 2,   // центр X
                dy + this.height / 2,  // центр Y
                r,                     // радіус
                0,                     // початковий кут
                Math.PI * 2            // повне коло (360°)
            );
            ctx.stroke();
            ctx.restore();
        }
        return; // не малюємо танк під час spawn-анімації
    }

    // Звичайне малювання танка (метод батьківського класу)
    super.render(ctx, ox, oy);
}
```

### Як працює мерехтіння?

```
flashFrame:  0  1  2  3  4  5  6  7  0  1  2  3 ...
Видно:       ✦  ✦  ✦  ✦  .  .  .  .  ✦  ✦  ✦  ✦ ...
```

`flashFrame < 4` — зірка видна перші 4 кадри, невидна наступні 4. Це створює плавне мигання.
