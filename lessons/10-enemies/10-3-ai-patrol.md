# 🤖 Урок 10.3: AI патрулювання

## Як рухаються вороги?

Вороги в Battle City мають простий але ефективний AI:

1. **Рухатись у поточному напрямку** — доки не зустрінеш перешкоду
2. **Кожні 0.8–2 секунди** — обрати новий випадковий напрямок
3. **Якщо застряг** (вперся в стіну) — негайно змінити напрямок

Цей простий алгоритм створює досить природний рух — вороги блукають по полю, іноді "випадково" рухаються до штабу або до гравця.

## 🧠 Метод update() — мозок ворога

Додаємо повний метод `update()` в Enemy:

```js
/**
 * Оновлення ворога кожен кадр — AI логіка
 * @param {number} dt - час між кадрами (мс)
 * @param {Function} canMove - (tank, nx, ny) => boolean
 * @param {number} now - поточний час (мс)
 */
update(dt, canMove, now) {
    // Мертвий ворог — нічого не робимо
    if (!this.alive) return;

    // ─── Spawn-анімація (мигаюча зірка) ──────────────────────
    if (this.spawnFlash) {
        // Зменшуємо таймер
        this.spawnTimer -= dt;
        // Збільшуємо кадр мерехтіння (0-7, по колу)
        this.flashFrame = (this.flashFrame + 1) % 8;
        // Коли таймер дійшов до 0 — зірка зникає, танк з'являється
        if (this.spawnTimer <= 0) {
            this.spawnFlash = false;
        }
        // Під час spawn-анімації ворог не рухається і не стріляє
        return;
    }

    // ─── Таймер зміни напрямку ───────────────────────────────
    // Збільшуємо лічильник часу
    this.dirTimer += dt;

    // Коли час вийшов — обираємо новий випадковий напрямок
    if (this.dirTimer >= this.changeDirIn) {
        this._pickRandomDir();
    }

    // ─── Спроба руху ─────────────────────────────────────────
    // move() повертає true якщо вдалося рухнутись, false якщо ні
    const moved = this.move(this.direction, dt, canMove);

    // Якщо не вдалося рухнутись — ворог застряг!
    if (!moved) {
        // Негайно змінюємо напрямок (не чекаємо таймер)
        this._pickRandomDir();
    }

    // ─── Стрільба ────────────────────────────────────────────
    // Ворог стріляє автоматично коли готовий кулдаун
    this.shoot(now, 'enemy');

    // Оновлюємо позиції куль ворога
    this.updateBullets();

    // ─── Оновлення кольору броньованого ──────────────────────
    // Якщо це броньований танк — колір змінюється з HP
    if (this.type === 'armor') {
        this.color = Enemy._colorByType('armor', this.hp, this.maxHp);
    }
}
```

## 🎲 Випадковий напрямок

```js
/**
 * Обирає новий випадковий напрямок руху
 * Скидає таймер і встановлює новий інтервал
 */
_pickRandomDir() {
    // DIRS = ['up', 'down', 'left', 'right']
    // Math.random() повертає число від 0 до 1
    // Math.floor(Math.random() * 4) дає 0, 1, 2 або 3
    this.direction = DIRS[Math.floor(Math.random() * DIRS.length)];

    // Скидаємо таймер
    this.dirTimer = 0;

    // Новий випадковий інтервал: від 800 до 2300 мс
    this.changeDirIn = 800 + Math.random() * 1500;

    // Вирівнюємо по сітці при повороті
    // (щоб танк не "застрягав" між тайлами)
    this.snapToGrid(this.direction);
}
```

### Чому `snapToGrid` при повороті?

Уявіть: танк рухається вправо і знаходиться між двома рядками тайлів. Якщо він раптом повернеться вгору — він буде між рядками і застряне! `snapToGrid` вирівнює позицію по найближчому тайлу перед поворотом.
