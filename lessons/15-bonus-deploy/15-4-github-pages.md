# 🌍 Бонус 15.4: Вмикаємо GitHub Pages

## Крок 1: Відкриваємо налаштування

1. Відкрий свій репозиторій: `https://github.com/ТВІЙ_ЛОГІН/battle-city`
2. Натисни вкладку **Settings** (шестерня зверху)
3. В меню зліва знайди розділ **Pages**

---

## Крок 2: Налаштовуємо Pages

На сторінці **GitHub Pages**:

1. В розділі **"Build and deployment"**:
   - **Source:** залиш "Deploy from a branch"
   - **Branch:** обери `main`
   - **Folder:** залиш `/ (root)`
2. Натисни **Save**

```
┌─────────────────────────────────────┐
│ Build and deployment                │
│                                     │
│ Source: Deploy from a branch   ▼    │
│                                     │
│ Branch: main ▼    / (root) ▼   Save │
└─────────────────────────────────────┘
```

---

## Крок 3: Чекаємо 1-2 хвилини

GitHub автоматично:
1. Бере файли з твого репозиторію
2. Публікує їх як вебсайт
3. Створює посилання

Через 1-2 хвилини поверни сторінку Pages (F5) — зверху з'явиться зелений банер:

```
✅ Your site is live at https://ТВІЙ_ЛОГІН.github.io/battle-city/
```

---

## Крок 4: Граємо!

Відкрий посилання — ти побачиш свою гру Battle City прямо в браузері!

Тепер можеш:
- Надіслати посилання друзям у месенджері
- Поділитись у соцмережах
- Додати в резюме як проєкт 😎

---

## Як оновити гру?

Якщо захочеш щось змінити (додати рівень, виправити баг), просто:

```bash
# Зміни файли в папці battle-city, потім:
git add .
git commit -m "Додав другий рівень"
git push
```

Через 1-2 хвилини сайт оновиться автоматично!
