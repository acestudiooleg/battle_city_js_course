# 🚀 Бонус 15.3: Завантажуємо на GitHub

## Крок 1: Створюємо репозиторій на GitHub

1. Відкрий [github.com/new](https://github.com/new)
2. **Repository name:** `battle-city`
3. **Description:** `NES Battle City гра на JavaScript`
4. Залиш **Public** (публічний — щоб GitHub Pages працював)
5. **НЕ** вмикай "Add a README file" (ми створимо свій)
6. Натисни **Create repository**

GitHub покаже сторінку з інструкціями — вони нам знадобляться!

---

## Крок 2: Ініціалізуємо Git в папці

Відкрий термінал в VS Code, перейди в папку `battle-city` і виконай:

```bash
# Ініціалізуємо Git — створюємо репозиторій на комп'ютері
git init
```

Команда `git init` створює приховану папку `.git` — це "база даних" Git, де зберігається вся історія.

---

## Крок 3: Додаємо файли

```bash
# Додаємо ВСІ файли до "зони очікування" (staging area)
git add .
```

Крапка `.` означає "все в поточній папці". Git тепер знає які файли потрібно зберегти.

Перевіримо:
```bash
git status
```

Має показати список зелених файлів — це файли які будуть збережені.

---

## Крок 4: Робимо коміт

```bash
# Зберігаємо (комітимо) всі додані файли з повідомленням
git commit -m "Battle City — моя перша гра на JavaScript!"
```

**Коміт** — це "знімок" всіх файлів. Як зберегти гру в іграх — можна повернутися до цього моменту.

---

## Крок 5: Підключаємо GitHub

Тепер з'єднаємо локальний репозиторій з GitHub. Заміни `ТВІЙ_ЛОГІН` на свій логін GitHub:

```bash
# Вказуємо куди відправляти код (origin = GitHub)
git remote add origin https://github.com/ТВІЙ_ЛОГІН/battle-city.git

# Перейменовуємо гілку в main (стандарт GitHub)
git branch -M main

# Відправляємо код на GitHub!
git push -u origin main
```

Може запитати логін і пароль від GitHub. Введи їх.

> **Підказка:** Якщо GitHub просить "Personal Access Token" замість пароля — створи токен: **Settings → Developer settings → Personal access tokens → Generate new token**. Обери scope "repo" і використай цей токен як пароль.

---

## Крок 6: Перевіряємо

Відкрий свій репозиторій у браузері:
```
https://github.com/ТВІЙ_ЛОГІН/battle-city
```

Ти маєш побачити всі свої файли! Якщо бачиш — ти молодець, код у хмарі!

---

## 📖 Шпаргалка команд Git

| Команда | Що робить |
|---------|-----------|
| `git init` | Створює репозиторій |
| `git add .` | Додає файли для коміту |
| `git status` | Показує стан файлів |
| `git commit -m "..."` | Зберігає зміни |
| `git push` | Відправляє на GitHub |
| `git pull` | Завантажує з GitHub |
