# 🌐 Бонус 15.1: Що таке Git і GitHub?

## Навіщо це потрібно?

Ти створив повноцінну гру — тепер покажи її друзям! Вони зможуть грати прямо в браузері, просто відкривши посилання.

Для цього нам знадобляться два інструменти:

### Git — система контролю версій

Уяви, що ти пишеш твір і хочеш зберегти кожну версію — чернетку, другий варіант, фінал. **Git** робить те саме для коду:

- Зберігає **історію змін** (можна повернутись до будь-якої версії)
- Дозволяє **завантажувати код** в інтернет
- Використовується **всіма програмістами** у світі

### GitHub — хмарне сховище

**GitHub** — це сайт, де зберігаються Git-репозиторії (проєкти). Як Google Drive для коду. Крім зберігання, GitHub має безкоштовну функцію **GitHub Pages** — вона перетворює твій проєкт у сайт з ігрою!

```
Твій комп'ютер          Інтернет
┌──────────┐    git push    ┌────────────┐    GitHub Pages    ┌──────────────┐
│ Файли    │ ──────────────► │  GitHub    │ ─────────────────► │ Сайт з грою  │
│ гри      │                │ репозиторій │                    │ battle-city  │
└──────────┘                └────────────┘                    └──────────────┘
```

## Крок 1: Створюємо акаунт на GitHub

1. Відкрий [github.com](https://github.com)
2. Натисни **Sign up** (зареєструватись)
3. Введи email, придумай пароль і логін (наприклад: `ivan-coder`)
4. Підтвердь email (перевір пошту)

Запам'ятай свій логін — він буде частиною адреси гри:
```
https://ivan-coder.github.io/battle-city/
```

---

## Крок 2: Встановлюємо Git

### Windows

1. Відкрий [git-scm.com/download/win](https://git-scm.com/download/win)
2. Завантаж інсталятор і запусти його
3. Натискай **Next** на кожному кроці (налаштування за замовчуванням підходять)
4. На кроці "Choosing the default editor" можеш обрати **Use Visual Studio Code**
5. Натисни **Install** і дочекайся завершення

### Mac

Відкрий термінал (Terminal) і введи:
```bash
git --version
```

Якщо Git ще не встановлений — macOS **автоматично** запропонує встановити його. З'явиться вікно "Install Command Line Developer Tools" — натисни **Install** і дочекайся завершення (може зайняти кілька хвилин).

Якщо вікно не з'явилось — введи вручну:
```bash
xcode-select --install
```

### Linux (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install git
```

---

## Крок 3: Перевіряємо що Git працює

Відкрий термінал в VS Code (`Ctrl + ~` або меню **Terminal → New Terminal**) і введи:

```bash
git --version
```

Має з'явитись щось на зразок:
```
git version 2.43.0
```

Якщо бачиш номер версії — Git встановлено! Якщо помилка — переглянь крок 2.

---

## Крок 4: Налаштовуємо Git

Git потрібно знати хто ти (ім'я та email). Це одноразове налаштування:

```bash
git config --global user.name "Твоє Ім'я"
git config --global user.email "твій@email.com"
```

Наприклад:
```bash
git config --global user.name "Ivan Petrenko"
git config --global user.email "ivan@gmail.com"
```

**Важливо:** email має бути такий самий як при реєстрації на GitHub!

Перевіримо:
```bash
git config --global user.name
git config --global user.email
```
