## 📐 Ключові розміри

| Елемент | Розмір | Опис |
|---------|--------|------|
| **Тайл (TILE)** | 16×16 px | Найменша одиниця — як піксель на карті |
| **Танк** | 32×32 px | 2×2 тайли — танк завжди займає 4 клітинки |
| **Ігрове поле** | 416×416 px | 26×26 тайлів — квадратне поле бою |
| **Рамка (BORDER)** | 16 px | Сіра рамка навколо поля |
| **Sidebar** | 160 px | Бічна панель справа з інформацією |
| **Canvas** | 608×448 px | Повний розмір вікна гри |

---

## 🧮 Як це рахується?

```
Canvas ширина = BORDER + FIELD_W + BORDER + SIDEBAR_W
608           = 16     + 416     + 16     + 160

Canvas висота = BORDER + FIELD_H + BORDER
448           = 16     + 416     + 16
```

---

## 🎯 Що таке "тайл"?

**Тайл** — це маленький квадрат 16×16 пікселів. Уяви собі клітинку в зошиті — ось це тайл! 📏

Ігрове поле складається з **26 рядків × 26 стовпців = 676 тайлів**.

Кожен тайл може бути:
- ⬛ **Порожнім** — танки їздять вільно
- 🧱 **Цеглою** — можна зруйнувати
- ⬜ **Бетоном** — не зруйнувати!
- 🌊 **Водою** — не можна проїхати
- 🌲 **Лісом** — танк ховається під деревами

А **танк** — це завжди 2×2 тайли (32×32 пікселі). Як 4 клітинки в зошиті, з'єднані разом!
