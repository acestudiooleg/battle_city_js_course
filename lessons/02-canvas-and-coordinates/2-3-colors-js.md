## 📝 Крок 2: Пишемо палітру

Відкрий `colors.js` і напиши:

```js
// === NES Battle City — Палітра кольорів ===

// 🖤 Базові кольори
export const black      = '#000000';  // Чорний (фон поля)
export const white      = '#fcfcfc';  // Білий (текст)
export const darkGray   = '#7c7c7c';  // Темно-сірий
export const lightGray  = '#bcbcbc';  // Світло-сірий

// 🖼️ Фон та рамка
export const fieldBg    = '#000000';  // Фон ігрового поля — чорний
export const borderBg   = '#5c5c5c';  // Рамка навколо поля — сіра

// 🟡 Гравець
export const playerYellow = '#f8f858';  // Жовтий танк гравця

// 🔴 Вороги (кожен тип має свій колір!)
export const enemyBasicColor  = '#e04038';  // Звичайний — червоний
export const enemyFastColor   = '#fcfcfc';  // Швидкий — білий
export const enemyPowerColor  = '#fcbc38';  // Потужний — помаранчевий

// 🛡️ Броня — кольори змінюються з кожним влучанням!
export const armorColors = ['#f8f858', '#fc7460', '#fcfcfc', '#7c7c7c'];

// 🧱 Матеріали стін
export const brickFull   = '#e04038';  // Цегла — червона
export const concreteCol = '#bcbcbc';  // Бетон — сірий
export const waterCol    = '#2038ec';  // Вода — синя
export const forestCol   = '#00a800';  // Ліс — зелений
export const iceCol      = '#bcfcfc';  // Лід — блакитний

// 🦅 Орел (штаб)
export const eagleAlive  = '#f8f858';  // Живий — жовтий
export const eagleDead   = '#7c7c7c';  // Знищений — сірий

// 📊 Sidebar (бічна панель)
export const sidebarBg   = '#5c5c5c';  // Фон панелі — сірий
export const sidebarText = '#fcfcfc';  // Текст — білий
```

## 🧠 Розбираємо кольори

### Формат `#RRGGBB`

Кожен колір записується як **6 символів після #**:

```
#e04038
 ││││││
 ││││└└── Blue  (38₁₆ = 56)  — трошки синього
 ││└└──── Green (40₁₆ = 64)  — трошки зеленого
 └└────── Red   (e0₁₆ = 224) — БАГАТО червоного!
```

Тому `#e04038` виглядає **червоним** — бо червоного найбільше!

### Чому NES кольори "не зовсім білі"?

```js
export const white = '#fcfcfc';  // а не #ffffff!
```

На NES не було "чистого білого" — найсвітліший колір був трошки тьмяніший. Ми зберігаємо це для автентичності!

## 💡 Навіщо окремий файл?

Тепер будь-який файл гри може **імпортувати** потрібні кольори:

```js
import { playerYellow, brickFull } from './colors.js';
```

Якщо дизайнер вирішить змінити колір танка — змінюємо **тільки в `colors.js`**, і весь проєкт оновиться!
