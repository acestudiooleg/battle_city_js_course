## 🎨 Що таке піксель-арт?

Піксель-арт — це стиль малювання де кожен піксель намальований вручну. На приставці NES (Nintendo Entertainment System) розрішення екрану було лише **256×240 пікселів**, тому кожен піксель важливий!

---

## 📦 Що таке спрайт-лист (Sprite Sheet)?

Спрайт-лист — це **одна велика картинка** що містить **всі маленькі картинки** гри:

```
┌──────────────┬──────────────┬────────────┐
│ x=0-127      │ x=128-255    │ x=256-399  │
│ Танки гравця │ Танки ворогів│ Тайли, UI  │
│ (жовті/зел.) │ (сірі/білі) │ Вибухи     │
└──────────────┴──────────────┴────────────┘
```

Наш файл `battlecity_general.png` має розмір **400×256 пікселів** і містить:
- **Танки гравців** (x: 0-127) — жовтий P1, зелений P2, різні рівні зірки
- **Танки ворогів** (x: 128-255) — 4 типи, кожен у 8 напрямках
- **Тайли** (x: 256+) — цегла, бетон, вода, ліс, лід
- **Орел** (штаб) — живий та мертвий стан
- **Вибухи, зірки, артефакти**

### Чому одна картинка а не 100 маленьких?

1. **Швидкість** — завантажити 1 файл швидше ніж 100
2. **Продуктивність** — браузер працює швидше з однією текстурою
3. **Зручність** — всі спрайти в одному місці

## 🔍 Як вирізати спрайт?

Уявіть спрайт-лист як **аркуш наклейок**. Кожна наклейка (спрайт) має координати `(x, y)` та розмір `(width, height)`.

Для малювання використовуємо `ctx.drawImage()` з 9 параметрами:

```js
ctx.drawImage(
    image,     // картинка-джерело (спрайт-лист)
    sx, sy,    // координати на листі (звідки вирізаємо)
    sw, sh,    // розмір вирізки (ширина × висота)
    dx, dy,    // куди малюємо на Canvas
    dw, dh     // розмір на Canvas (можна масштабувати!)
);
```

### Приклад: вирізаємо танк

```
Спрайт-лист:                    Canvas:
┌─────────────────┐              ┌───────────────┐
│ (0,0)  ┌──┐     │              │               │
│        │🔵│ 16×16│    ═══►     │  ┌────┐       │
│        └──┘     │              │  │ 🔵 │ 32×32 │
│                 │              │  └────┘       │
└─────────────────┘              └───────────────┘

ctx.drawImage(img,  0, 0, 16, 16,  100, 200, 32, 32);
//                  ─── джерело ──  ── призначення ──
```

Спрайт 16×16 на листі → малюється як 32×32 на Canvas (збільшено вдвічі).

## ⚠️ Важливо: imageSmoothingEnabled

При збільшенні піксель-арту браузер за замовчуванням **згладжує** пікселі, що робить картинку розмитою. Потрібно вимкнути це:

```js
ctx.imageSmoothingEnabled = false;
```

Без цього рядка:
```
Оригінал: ■□■    Згладжене: ░▒░
          □■□                ▒█▒
          ■□■                ░▒░
```

З `imageSmoothingEnabled = false`:
```
Оригінал: ■□■    На Canvas: ██  ░░  ██
          □■□               ░░  ██  ░░
          ■□■               ██  ░░  ██
```

Чіткі пікселі — саме те що нам потрібно!
