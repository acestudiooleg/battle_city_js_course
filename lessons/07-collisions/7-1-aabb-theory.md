# 🎮 Урок 7.1: Теорія AABB — перевірка перетинань

## Як перевірити зіткнення? 💥

## Що таке AABB?

**AABB** = Axis-Aligned Bounding Box (прямокутник, вирівняний по осях).

Простіше: це звичайний прямокутник без обертання, який описується 4 числами:
- `x, y` — лівий верхній кут
- `width, height` — ширина та висота

```
(x, y)
  ┌─────────────┐
  │             │
  │    AABB     │  height
  │             │
  └─────────────┘
       width
```

---

## Формула перетинання

Два прямокутники **перетинаються**, якщо виконуються **всі 4 умови**:

```
A.x < B.x + B.width     (лівий край A лівіше правого краю B)
A.x + A.width > B.x     (правий край A правіше лівого краю B)
A.y < B.y + B.height    (верхній край A вище нижнього краю B)
A.y + A.height > B.y    (нижній край A нижче верхнього краю B)
```

В коді:
```javascript
function aabb(ax, ay, aw, ah, bx, by, bw, bh) {
    return ax < bx + bw && ax + aw > bx && ay < by + bh && ay + ah > by;
}
```

---

## Візуальна перевірка

```
Перетинаються (✅):          Не перетинаються (❌):

┌─────┐                     ┌─────┐
│  A  │                     │  A  │
│   ┌─┼───┐                └─────┘
└───┼─┘   │                         ┌─────┐
    │  B  │                         │  B  │
    └─────┘                         └─────┘
```

---

## Де це використовується?

| Зіткнення | Що перевіряємо |
|-----------|----------------|
| Куля ↔ Стіна | Куля влучила в цеглу? → руйнуємо! |
| Куля ↔ Танк | Куля влучила в ворога? → знищуємо! |
| Танк ↔ Стіна | Танк не може проїхати крізь стіну |
| Танк ↔ Танк | Танки не проходять одне через одного |
| Куля ↔ Куля | Кулі знищують одна одну |

---

## Підсумок

- ✅ AABB — прямокутник з x, y, width, height
- ✅ Два AABB перетинаються, якщо виконуються 4 умови
- ✅ Одна функція `_aabb()` використовується для всіх типів зіткнень
